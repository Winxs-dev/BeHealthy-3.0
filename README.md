# BeHealthy 

Um web app simples e responsivo para ajudar a manter hábitos saudáveis.

Funcionalidades principais:
- Login e cadastro com Firebase
- Cálculo de meta diária de água (baseado no peso) + registro de consumo com histórico
- Cálculo de IMC
- Galeria de receitas fitness com filtros
- Newsletter no footer (envia inscrições por e-mail)

Tecnologias:
- HTML, CSS e JavaScript puro
- Firebase (Authentication + Firestore)
- EmailJS (newsletter)
